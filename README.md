# Institute Management System

A simple Python project for managing students, courses, and enrollments in an institute.

## Features

- CRUD operations for students, courses, and enrollments
- In-memory storage

## Installation

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the demo: `python main.py`
4. Run tests: `python tests/test_crud.py`

## Structure

- `src/models.py`: Data models
- `src/crud.py`: CRUD operations
- `main.py`: Demo script
- `tests/`: Unit tests
